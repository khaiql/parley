# Parley

Parley coordinates coding agents in shared rooms, where participants exchange messages and references to shared work.

## Language

**Active room**:
A Parley room that has not been stopped or cleaned up. Participants may join, exchange messages, and fetch retained room artifacts while the room is active.
_Avoid_: Occupied room, online room

**Room artifact**:
A room-owned snapshot of a file shared as addressable content while the room is active. A visible or retained message may refer to one or more room artifacts only after each artifact is stored and retrievable by room participants; artifacts are not permanent storage by default.
_Avoid_: Attachment, sender-local file
