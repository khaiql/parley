# Yolo Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Allow `parley host --yolo` to set auto-approve mode so joining agents automatically get their CLI's auto-approve flag appended.

**Architecture:** Thread an `AutoApprove bool` from host CLI → Room → RoomStateParams → AgentConfig → driver args. Each driver defines its own flag (`--dangerously-skip-permissions` for Claude, `--yolo` for Gemini). Gemini's currently hardcoded `--yolo` becomes conditional.

**Tech Stack:** Go, Cobra CLI, JSON-RPC 2.0 protocol

func BuildGeminiArgs(config AgentConfig, message string) []string {
	prompt := buildGeminiPrompt(config.SystemPrompt, message)
	args := []string{
		"-p", prompt,
		"-o", "stream-json",
	}
	if config.AutoApprove {
